# PhasmoWiki 👻

**PhasmoWiki** — це комплексний веб-додаток, створений для допомоги гравцям у грі Phasmophobia. Тут ви знайдете всі необхідні інструменти для успішного полювання на привидів, зручний довідник зі спорядження та інтегрованого ШІ-асистента.

## ✨ Основний функціонал

### 1. Визначник привидів (Ghost Identifier)
Цей інструмент дозволяє швидко звузити коло підозрюваних привидів на основі знайдених доказів у грі.
- **Як використовувати:** Просто відмічайте докази, які ви знайшли (наприклад, ЕМП 5, Радіоприймач, Записи в блокноті), і додаток автоматично відфільтрує список можливих привидів, щоб показати вам, з ким ви маєте справу.

### 2. Спорядження (Equipment)
Повний довідник по всьому доступному спорядженню в грі.
- **Як використовувати:** Перейдіть на вкладку "Спорядження", щоб переглянути список предметів, дізнатися як вони працюють і для чого використовуються. Додаток отримує зображення екіпірування напряму з GitHub сховища.

### 3. ШІ Асистент (AI Assistant)
Ваш особистий розумний помічник, який знає все про гру.
- **Як використовувати:** Натисніть на іконку чату в правому нижньому куті (або на відповідну вкладку), щоб відкрити віджет ШІ асистента. Ви можете задати йому будь-які питання про привидів, тактики виживання, роботу обладнання тощо.

### 4. Панель Адміністратора
Для підтримання бази знань в актуальному стані передбачена закрита секція адміністратора.
- **Як використовувати:** Доступна тільки для авторизованих користувачів з правами адміністратора. Дозволяє додавати, редагувати та видаляти інформацію про привидів та спорядження в реальному часі.

## 🚀 Як розпочати роботу

Цей додаток має зручний користувацький інтерфейс, стилізований під темну атмосферу гри з ефектами напівпрозорого скла (Glassmorphism).

1. **Авторизація:** Ви можете увійти за допомогою свого облікового запису Google. Для цього натисніть кнопку "Увійти через Google" у верхньому правому куті сайту. Деякі функції (наприклад, ШІ 아систент) можуть вимагати авторизації.
2. **Навігація:** Використовуйте верхнє меню для швидкого перемикання між інструментами. 
3. **Адаптивність:** Додаток чудово працює як на ПК під час гри, так і на мобільних пристроях, якщо ви хочете тримати телефон поруч як додатковий дисплей зі списком доказів.

## 🛠 Технології
- **Фронтенд:** React, TypeScript, Tailwind CSS
- **Бекенд/АПІ:** Express.js (Node.js) для проксіювання запитів до бази даних і Gemini API
- **База даних:** Supabase (PostgreSQL) для збереження інформації 
- **ШІ:** Google Gemini API
