<h1>PhasmoWiki 👻</h1>

<p><strong>PhasmoWiki</strong> — це комплексний веб-додаток, створений для допомоги гравцям у грі Phasmophobia. Тут ви знайдете всі необхідні інструменти для успішного полювання на привидів, зручний довідник зі спорядження та інтегрованого ШІ-асистента.</p>

<h2>✨ Основний функціонал</h2>

<h3>1. Визначник привидів (Ghost Identifier)</h3>
<p>Цей інструмент дозволяє швидко звузити коло підозрюваних привидів на основі знайдених доказів у грі.</p>
<ul>
  <li><strong>Як використовувати:</strong> Просто відмічайте докази, які ви знайшли (наприклад, ЕМП 5, Радіоприймач, Записи в блокноті), і додаток автоматично відфільтрує список можливих привидів, щоб показати вам, з ким ви маєте справу.</li>
</ul>

<h3>2. Спорядження (Equipment)</h3>
<p>Повний довідник по всьому доступному спорядженню в грі.</p>
<ul>
  <li><strong>Як використовувати:</strong> Перейдіть на вкладку "Спорядження", щоб переглянути список предметів, дізнатися як вони працюють і для чого використовуються. Додаток отримує зображення екіпірування напряму з GitHub сховища.</li>
</ul>

<h3>3. ШІ Асистент (AI Assistant)</h3>
<p>Ваш особистий розумний помічник, який знає все про гру.</p>
<ul>
  <li><strong>Як використовувати:</strong> Натисніть на іконку чату в правому нижньому куті (або на відповідну вкладку), щоб відкрити віджет ШІ асистента. Ви можете задати йому будь-які питання про привидів, тактики виживання, роботу обладнання тощо.</li>
</ul>

<h3>4. Панель Адміністратора</h3>
<p>Для підтримання бази знань в актуальному стані передбачена закрита секція адміністратора.</p>
<ul>
  <li><strong>Як використовувати:</strong> Доступна тільки для авторизованих користувачів з правами адміністратора. Дозволяє додавати, редагувати та видаляти інформацію про привидів та спорядження в реальному часі.</li>
</ul>

<h2>🚀 Як розпочати роботу</h2>

<p>Цей додаток має зручний користувацький інтерфейс, стилізований під темну атмосферу гри з ефектами напівпрозорого скла (Glassmorphism).</p>

<ol>
  <li><strong>Авторизація:</strong> Ви можете увійти за допомогою свого облікового запису Google. Для цього натисніть кнопку "Увійти через Google" у верхньому правому куті сайту. Деякі функції (наприклад, ШІ асистент) можуть вимагати авторизації.</li>
  <li><strong>Навігація:</strong> Використовуйте верхнє меню для швидкого перемикання між інструментами.</li>
  <li><strong>Адаптивність:</strong> Додаток чудово працює як на ПК під час гри, так і на мобільних пристроях, якщо ви хочете тримати телефон поруч як додатковий дисплей зі списком доказів.</li>
</ol>

<h2>🛠 Технології</h2>
<ul>
  <li><strong>Фронтенд:</strong> React, TypeScript, Tailwind CSS</li>
  <li><strong>Бекенд/АПІ:</strong> Express.js (Node.js) для проксіювання запитів до бази даних і Gemini API</li>
  <li><strong>База даних:</strong> Supabase (PostgreSQL) для збереження інформації</li>
  <li><strong>ШІ:</strong> Google Gemini API</li>
</ul>
